Final Project Brief

Theme: Stardew Valley Madlibs Skit

I will be updating my first project, Madlibs 2.0 for my final project because there is much more I can do to improve its user experience and allow for stronger game immersion for the user.

Project Description

For my final project, I will be improving my Stardew Valley themed Madlibs game. I will be focusing on animations and overall details of the game's actions to make the user experience more interesting.

The goal will be to make this Madlibs imitate the aesthetics of the Stardew Valley game. Overall, the main improvements will be focused on the design, and some usability improvements will be included as well.

Specifics

For animations, my plan will be to add movement in the start screen. I will insert clouds in the background and have them move across the screen at differente paces.

I will add a sound feature for each time the user clicks on a button.

I will add a back button feature and a restart game feature to allow the user to backtrack if they have made any actions they want to redo.

I plan to fix the bugs in my code as well. There are issues such as awkward background image sizing and misaligned overlays. There are also errors in the .hidden and .showing CSS features.
