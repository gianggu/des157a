Tasks Tested

  1. Starting the game
  2. Filling out answers
  3. Submitting the answers
  4. Restarting the game

Critiques

Strong Points:

The overall balance of the design is cohesive and emits the vibe of the game being referenced (Stardew Valley).

There is good contrast between the backgrounds and the text.

The hover states are a good touch to make possible actions for users more obvious.

Users have the option to go back to different stages of the game.

Weak Points and Improvements:

The music button is not noticable as it is now and there is no indiciation for the mode it is on (on/off).

The restart button and back button should switch locations for better usability.

The restart button is only appearing on one overlay but it should be on two of the overlays.

The start screen's "Stardew Madlibs" button is not as intuitive to press. Consider adding a button or some feature that would make it more obvious to users that it should be pressed.

There is some space that makes the screen scrollable even though there is no content which could cause confusion to the users.
