Madlibs Updated Prototype

Images not uploaded due to updates mostly being animations.

What I have done is add animations to my Madlibs to make it more immersive and less visually static. In the start screen, clouds that I illustrated will be floating across the screen at different speeds. This makes the start screen more interesting, and adds value to the whole game and not just to the text input modal. I also made the modal dissolve transitions more seamless.

Another aspect I focused on was the usability in addition to the visual design. For this, I added a back button in all following steps after the start screen. This always the users more control over their actions and any changes they want to make. I also added a Return to Start button so users don't need to refresh the page to play again.
